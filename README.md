# 每日完成清单（Daily Checklist）

一个纯前端、可「安装到手机主屏」的每日完成清单 PWA。

- 任务与完成情况只存在你打开它的那台设备（浏览器 localStorage），**不上传、不进任何服务器、也不进本仓库**。
- 功能：打勾核对、完成进度、连续天数、最近 14 天历史、按餐次服药提醒、明暗主题。
- 「导出 / 导入备份」：把数据存成 JSON 文件，留在你自己的手机或云盘；误清浏览记录后用「导入备份」恢复。

## 部署
纯静态站点，由 Cloudflare Pages 从本私有仓库自动发布。

## 隐私说明
本仓库只包含应用代码，**不含任何个人数据**。你的清单数据始终保存在你本地设备上。
